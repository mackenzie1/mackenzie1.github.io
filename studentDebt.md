## Student Debt Calculator
# Purpose:
This project was created to estimate the price of college if you are a future student. It takes the number of years until college and then the number of years in college to calculate how much college will cost you in the future. 
![image](https://user-images.githubusercontent.com/49001843/87089615-ad37e280-c204-11ea-8cb1-28de389067f1.png)
# How it works:
This project was built in java and uses four different classes: StudentDebtCalculator, User, Menu, and Calculation. The StudentDebt file is the main drive class. The user file creates a user class with everything we need for a log in and password. We create the user object in the main class. The Menu class is simply a class with on method showMenu() and prints out the options and accepts an int as the return. The calculation class was the most challenging. In this class we handle all the calculations and the majority of the functionality. I tested with different formulas and translated them into java code. To check and see if the calculation was correct I used tools online that preformed similar calculations.
